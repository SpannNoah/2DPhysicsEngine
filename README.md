# 2D Physics Engine

A 2D physics engine built in C++ with SDL2, created for educational purposes while following [Gustavo Pezzi's](https://pikuma.com/) 2D game physics programming course.

## Features (In Progress)

- Custom `Vec2` math library with vector operations
- Rigid body simulation with shape support (circle, box, polygon)
- Force generators (gravity, drag, spring, etc.)
- SDL2 rendering with texture support

## Building

This project uses Visual Studio and SDL2 on Windows.

### Prerequisites

- Visual Studio (with C++ Desktop Development workload)
- SDL2 and SDL2_image (included in the repo)

### Steps

1. Open `2DPhysicsEngine.sln` in Visual Studio
2. Build and run (F5)
